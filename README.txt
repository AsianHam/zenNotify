The purpose of this program is to notify a Windows 10 user if a new ticket has rolled in the Zendesk ticketing system.

The python file is meant to run indefinitely once started; the program can be occassionally restarted to clear the cache.

THe notification system has roughly a one minute delay between the ticket actually appearing and the system notifying you.
